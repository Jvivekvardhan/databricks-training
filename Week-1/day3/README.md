# SQL Joins Assignment

This assignment focuses on practicing different types of SQL joins such as:

- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- CROSS JOIN

## Topics Covered

### 1. LEFT JOIN
Used to return all rows from the left table and matching rows from the right table.

### 2. RIGHT JOIN
Used to return all rows from the right table and matching rows from the left table.

### 3. FULL OUTER JOIN
Used to return all matching and unmatched rows from both tables.

### 4. CROSS JOIN
Used to generate every possible combination of rows between two tables.
